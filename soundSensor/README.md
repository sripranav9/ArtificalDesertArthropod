## SparkFun Microphone Breakout w/ Potentiometer

**Materials Needed**
* Feather M4 Express
* SparkFun Microphone Breakout
* Potentiometer
* 7 jumper wires
* Resistor (10k ohm)
* Power Source (Laptop or Battery)

**Microphone Connections**
1. 3V - VCC
2. GND - resistor - gnd (microphone)
3. A0 - AUD

**Potentiometer Connections**
1. A3 - potentiometer
2. GND - potentiometer
3. 3V - potentiometer
